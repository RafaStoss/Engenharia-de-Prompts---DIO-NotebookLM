📑 Mini Guia de Estudos
A Engenharia de Prompts é a prática de criar instruções claras e objetivas para orientar modelos de IA.  
- Quanto mais específico o prompt, maior a qualidade da resposta.  
- Técnicas como Zero-Shot, Few-Shot e Chain of Thought ajudam a reduzir alucinações.  


📚 Glossário
- Prompt: Instrução fornecida à IA para realizar uma tarefa. 
- LLM (Large Language Model - Modelo de Linguagem de Grande Porte): Modelo de IA treinado para compreender e gerar textos semelhantes aos produzidos por seres humanos. 
- Tokens: Unidades básicas de texto processadas pela IA. 
- Engenharia de Prompts: Prática de criar e aperfeiçoar prompts.  
- Componentes de um Prompt: Instrução, contexto, exemplos e formato de saída.
- Técnicas de Engenharia de Prompts: Zero-Shot, Few-Shot, Chain of Thought etc.  
- Alucinação de IA: Informações incorretas ou inventadas geradas pela IA.
